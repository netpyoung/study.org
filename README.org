** 왜 Org ?

- emacs를 메인 편집기로 사용하고 있다.
- markdown의 사용에 한계를 느꼈다.
  - 좀 더 기능이 많은 org로 눈길을 돌리기 시작.
- 어떻게 다른 사람에게 알릴까?
  - 대상 타겟을, emacs를 다룰 줄 알며, markdown을 알고 있는 사람을 타겟으로 하자.
